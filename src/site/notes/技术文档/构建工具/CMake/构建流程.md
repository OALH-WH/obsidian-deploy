# 基本步骤
## 配置项目
```shell
# eg.1 老版本配置, 常见
cd ./build && cmake ..

# eg.2 新版本配置, 需要支持-S and -B参数
cmake -S ./src -B ./build
```
## 构建项目
```shell
cmake --build ./build
```
## 安装项目(可选)
```shell
cmake --install ./build
```
