# 使用
- 高亮显示的字母是快捷键
- 键入`y`选择一个选项
- 当选项为`N`表示排除这个选项
- 键入两次`ESC`退出
- 键入`?`, 显示帮助文档
- 键入`/`来**搜索**
- 当选项为\*表示选项被选中
- 当写文本框时, `ctrl+`键入删除, 才能删除默认文本
## 搜索选项
### 内容解析
- `Location`: 此选项所在路径
- `(NUMBER)`: 键入对应的数字可直接跳转到其位置
- `Depends`: 此选项有哪些依赖和依赖之间的关系
- `Selects`: 此选项使能会自动选中哪些选项
- `Selected by`: 哪些选项选中会自动选中此选项
