# 芯片
- `CPU, Central Processing Unit`: 中央处理单元
- 只有`CPU`是无法工作的
- 可访问的地址线宽度: 32位, 64位...
	- 表示可访问的地址数量, 每一个地址有1个字节的空间
	- 因此比如32位的芯片, 最多能访问的存储空间为`2^32 / 2^30 = 2^2 = 4GB`
# MCU
- 包含了`CPU`, 内存, `Flash`, 各种外设的完整芯片
# PC
- 包含了`CPU`, 内存条, 硬盘, 各种外设的完整主板
# 中断向量表
- 向量表的结构由ARM规范, 向量表起作用的位置由芯片厂商决定, APP向量表的存放位置由开发者根据链接脚本决定
# 关系与区别
## PC和MCU的区别
- MCU集成度高, 最后被芯片厂商做成了一颗完整芯片(黑盒)
- PC是模块化的, 通过一个主板, 把CPU和各个外设放到一个完整主板上
## 主机电脑
