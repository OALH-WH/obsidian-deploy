# 写缓存
**原理**
把要写入的地址和值存储起来, 多条写入的命令一次性通过`AHB/AXI`总线发送给内存/外设, 有以下缺点:
- 内存/外设知道要写入的地址和值, 但发生错误时, 无法回传信息帮助定位错误, 只会发送错误bit位, 告诉`CPU`发生了总线错误
- `CPU`此时只能上报错误的类型, 比如总线错误, 以及可以通过`SCB->`

# 中断向量表
1. **中断向量表的结构** → **由 ARM 决定**（架构规范）
2. **中断向量表在芯片内部的具体存放位置** → **由链接脚本决定**（开发者/工程配置）
3. **这个位置能否“起作用”（即 CPU 能否正确找到它）** → **由芯片厂商决定**（硬件地址映射）
