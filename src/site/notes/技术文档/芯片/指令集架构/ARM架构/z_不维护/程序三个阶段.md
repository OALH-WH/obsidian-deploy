1. 示例

1. 链接脚本

```cpp
MEMORY
{
    ROM (rx) : ORIGIN = 0x00000000, LENGTH = 1M
    RAM (rw) : ORIGIN = 0x30000000, LENGTH = 16M
}

SECTIONS
{
    .text : {
        *(.text)
    } > ROM

    .data : {
        *(.data)
    } > RAM AT> ROM

    .bss : {
        *(.bss)
    } > RAM
}
```

1. 程序存储阶段（使用LMA）

1. 程序存储在非易失性存储器中，.text段的加载地址是0x0000_0000, .data段的加载地址也是0x0000_0000，如果多个段的LMA相同，烧录工具会按照顺序烧录各个段，不会发生程序烧录冲突

1. 程序加载阶段（使用VMA）

1. 启动代码将程序从存储器加载到内存，根据链接脚本将各个段放在运行时地址上

1. 程序运行阶段（使用VMA）

1. CPU从0x0000_0000开始执行指令，按照程序的运行时地址运行程序
