# 常用函数
## `VLOOKUP`
- 作用: 查找某一个单元格所在行对应的哪一列的内容
- 参数
	- arg1: 要查找的值
	- arg2: 到哪里找
		- 哪个表!哪从哪个列开始:到哪个列结束
	- arg3: 返回第几列的数据
		- 从开始的列开始算, 开始的列是1
	- arg4: 匹配模式
- 实例
	- `=VLOOKUP(A2, Sheet1!A:D, 2, FALSE)`
- 
1. 语法

2. 函数

3. ROW: 获取当前行号

4. COLUMN: 获取当前列号

5. INDIRECT：参数为单元格地址，返回值为单元格的引用，效果类似D2这种

6. ADDRESS：根据行号和列号返回单元格地址

7. OFFSET

8. 示例：OFFSET(D2, 0, -1, 20, 1) //D2是当前单元格，0是行偏移量，0表示不偏移，-1是列偏移量，表示向左偏移1列，20表示范围，即{2~21}行，返回值和C$2:C$21相同

9. INDEX：根据列范围or行范围和范围索引来获取指定单元格的值

10. 示例：INDEX(C$1:C$21,ROW()) //根据C列{1~21列}这个数组范围的索引（通过row函数来获取行号，1~21列索引号就是行号，如果是2~21列，索引号=行号-1）来获取指定值

11. MATCH

12. 示例：MATCH(C2,A$2:A$21,0) //根据单元格C2的值，查找A列{2~21}行单元格的值，0代表精确匹配，即=，返回从第2行开始，第几行匹配到的（比如在A2找到了，返回1，A3找到了，返回2）
