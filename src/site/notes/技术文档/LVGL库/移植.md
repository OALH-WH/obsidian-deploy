# 配置
## 创建配置头文件
**`lv_conf.h`**
可以把`lvgl/lv_conf_template.h`中的示例配置复制到`lv_conf.h`中

将文件开头的`#if 0`改为`#if 1`, 并通过`LV_COLOR_DEPTH`宏定义, 修改屏幕颜色深度
## 文件基本布局
```shell
lvgl/
lv_conf.h
other files and folders in your project
```
## 文件布局调整
**`lv_conf.h`**
如果需要将该文件移动到其他位置, 需要向编译器选项中添加`-DLV_CONF_INCLUDE_SIMPLE`, 并且手动设置包含路径`-I/your/path`

如果需要改文件名称, 则需要向编译器添加选项`-DLV_CONF_PATH=/your/path/file`, 如果添加了这个宏, 则默认`LV_CONF_SKIP`为`0`

如果`LV_CONF_SKIP`为`0`, `LVGL`将不会尝试包含`lv_conf.h`

# 连接硬件
## 初始化`LVGL`
1. 在系统执行的早期调用`lv_init()`对`LVGL`初始化
2. 初始化驱动程序
3. 连接`tick_interface`
	- 目的是为了执行定时任务 
4. 连接`Display Interface`
	- 目的是为了显示内容
5. 连接`Input-Device Interface`
	- 目的是为了接受点击事件
6. 每隔几秒调用一次`lv_timer_handler`
	- 目的是为了统一执行`LVGL`跟时间相关的任务
