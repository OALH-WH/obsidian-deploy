# hasattr(函数名, 属性名称字符串)
- 用于判断函数属性是否存在
# enumerate()
- 用于`python`数据结构的迭代
## 返回值
- `(0, seq[0]), (1, seq[1]), (2, seq[2]), ...`
---
1. all()：判断一个列表或集合所有元素是否满足条件，有一个不满足返回false

2. any(): 判断一个列表或集合任意元素是否满足条件，有一个满足返回true

3. excel(): 作用下同

4. eval()：将字符串作为代码执行

5. 实例1

```cpp
# 使用 eval
x = 10
result = eval("x + 5")  # 将字符串作为表达式执行
print(result)  # 输出: 15

# 使用 exec
var_name = "y"
exec(f"{var_name} = 300")
```
