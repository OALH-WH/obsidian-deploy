1. 实例

1. 快速把数据导出为excel

1. 数据集合格式

```python
data1 = {
    '姓名': ['张三', '李四', '王五'],
    '年龄': [25, 30, 35],
    '城市': ['北京', '上海', '深圳']
}

data2 = {
    '课程': ['语文', '数学', '英语'],
    '分数': [90, 80, 70]
}
```

1. 数据导出方式

```python
df1 = pd.DataFrame(data1)
df2 = pd.DataFrame(data2)

filepath = r'D:\write_to_excel.xlsx'
df1.to_excel(filepath, index=False)

```
