# 速查表
| 方法                         | 核心对象/函数     | 如何获取捕获组内容                    | 主要特点与适用场景                                                                             |
| -------------------------- | ----------- | ---------------------------- | ------------------------------------------------------------------------------------- |
| **`match()` 或 `search()`** | `match` 对象  | `.group(n)`  <br>`.groups()` | **匹配单次**。`match`从开头匹配，`search`搜索整个字符串。                                                |
| **`findall()`**            | 直接返回列表      | 列表中的**元组**或**字符串**           | **匹配所有**。当模式**只有1个**捕获组时，列表元素是**字符串**；当有**2个及以上**组时，元素是**包含各组的元组**。**不保留`group(0)`**。 |
| **`finditer()`**           | 迭代`match`对象 | 对每个`match`对象调用`.group(n)`    | **匹配所有**，且能获取每个匹配的**全部信息**（包括`group(0)`和各捕获组），性能更优。                                   |
| **`split()`**              | 返回列表        | 列表中的**分隔符部分**                | 用捕获组**模式作为分隔符**时，**捕获组本身匹配到的内容**也会被保留在结果列表中。                                          |
