# make
## 常用make目标
- `make`: 构建`bootloader`(可选), `linux kernel`(可选), `root file system`(可选)
	- `bootloader`输出目录
- `make help`: 查看所有支持的make目标
### busybox
- `make busybox-menuconfig`: 打开busybox软件包的配置界面, 具体配置参考[[技术文档/Linux系统常见构建工具/BusyBox/使用\|使用]]
- `make busybox`: 构建busybox的配置
### 杂项/其他(miscellaneous)
- `make show-info`: 生成一个json文本输出关于所有包的信息
- `make show-targets`: 查看所有配置了的目标软件包
