# 关键组件
-  内置`BusyBox`
## `Makefile`和 `Kconfig`构建工具

# 定制性
- 通过菜单选择或者覆盖文件修改包
- 菜单中带BR的选项的意思就是GNU标准
# 输出产物
- 一个可直接烧写的完整系统镜像(内核+根文件系统)
# 开发模式
- 和 `Yocto`属于同一类别的嵌入式Linux系统构建工具, 参考[[技术文档/Linux系统常见构建工具/Yocto/背景\|背景]]
