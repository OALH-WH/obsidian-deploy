- **与 Debian 的关系**：Buildroot/Yocto 和 Debian 是 **“制造厂”与“成品超市”** 的关系。你用前者从零开始定制一辆专属汽车；后者则是直接提供一辆功能齐全、维护良好的品牌汽车。一个典型的组合是，用 Yocto 为嵌入式设备构建基础系统，后期再用 `apt` 从 Debian 仓库安装高级应用。
    
- **与 BusyBox 的关系**：BusyBox 是 Buildroot 和 Yocto 构建的**系统中最核心的组件之一**。在构建菜单中，你可以选择使用完整版的 GNU Coreutils，还是精简版的 BusyBox 来提供系统命令。绝大多数嵌入式镜像都选择 BusyBox 以节省空间。

`Yocto`和`Buildroot`都支持生成`sdk`包
