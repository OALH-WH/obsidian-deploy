1. 原理

- 虚拟机与宿主机同网络：在桥接模式下，虚拟机的网络接口通过虚拟化软件（例如 VirtualBox、VMware）与宿主机的物理网络接口直接连接(相当于虚拟机和主机公用一个网络接口)。这使得虚拟机与宿主机处于同一个网络环境中。

- 虚拟机的 IP 地址：虚拟机通常会通过 DHCP 从物理网络（例如路由器或 DHCP 服务器）获取一个与宿主机相同子网的 IP 地址。例如，如果宿主机 IP 是 192.168.1.10，那么虚拟机可能会获取到 192.168.1.20。

- 虚拟机与外部通信：虚拟机可以像宿主机一样直接与外部网络（包括互联网）通信。它与宿主机在同一个网络中，直接由网络中的路由器、交换机等设备管理和转发数据。

- 外部访问虚拟机：由于虚拟机与宿主机在同一局域网内，其他物理设备可以直接访问虚拟机。虚拟机可以充当 Web 服务器或其他类型的服务器，外部网络设备可以直接访问虚拟机的 IP 地址。
