# 原理
简单说，它是一小段运行在 MCU RAM 里的代码，负责接收调试器（如 J-Link、ST-Link）发来的数据，并按照 Flash 的时序要求完成擦除和写入。  
对于你用的 W25Q，它就是一个 **.FLM 文件**（Keil）、.elf 文件（IAR/SEGGER）或 OpenOCD 脚本，告诉工具：“要把固件烧到外部 Flash，就执行这个”。

## 寻找方法

# 使用
通过调试器加载算法文件到MCU RAM里, 然后根据调试器指令, 烧录数据
