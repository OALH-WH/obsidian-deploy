# 远程拉取分支到本地
- `svn checkout <URL>`
	- 拉取分支到当前目录, 包括根目录
---
1. 保存差异文件

2. svn diff > .../....patch

3. 恢复差异

4. patch -p0 .../....patch

5. 把另外一个分支（url）的所有提交合并到当前分支

6. svn merge url  -r 0:HEAD

7. 本地代码降版本到指定版本号

8. svn up -r 版本号

9. 回滚代码到当前版本

10. svn revert -R .

11. 撤销某个分支的改动

12. svn merge -c -<版本修订号> .

13. 示例：svn merge -c -32280 .

14. 回退之前的版本覆盖最新版 

15. svn merge -r 100:90 .

16. svn commit

17. 修复svn本地错误

18. svn cleanup 本地分支路径

19. 拉取新的分支里的指定版本到另一个分支中

20. svn copy 

21. 拉取分支

22. svn库中创建一个文件夹

23. cp其他分支的代码到这个文件夹中

24. svn cp url .
