# 删除
## 方法一：通过 GitLab Web 界面删除（最常用）

这是最直观、推荐的方式
1. **进入项目**：在 GitLab 页面上，进入你想要删除的项目主页。
2. **打开设置**：在左侧边栏中，点击 **设置 (Settings)** > **通用 (General)**。
3. **展开高级设置**：在页面中找到并点击 **展开高级 (Expand Advanced)** 部分。
4. **删除项目**：在页面底部，找到 **删除项目 (Remove project)** 区域。
5. **确认删除**：
    - 在弹出的确认框中，根据提示输入**项目名称**以进行确认。
    - 点击 **确认 (Confirm)** 按钮。

> **注意**：为防止误删，GitLab 可能默认启用了“延迟删除”功能。项目可能会被标记为待删除，并在几天后（默认7天）才被永久移除[](https://docs.gitlab.co.jp/ee/user/group/manage.html)。在此期间，可以在项目设置中恢复它。
