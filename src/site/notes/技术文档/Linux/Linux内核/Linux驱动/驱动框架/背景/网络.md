# 有线网络
需要`MAC`和`PHY`两个芯片实现上网
## 嵌入式网络硬件方案
### SOC + (MAC+PHY)
即`SOC`不内置`MAC`芯片, 需要外接`MAC+PHY`一体的网络芯片

这种方案的优点就是让不支持网络的 SoC 能够另辟蹊径，实现网络功能，但是缺点就是网络效率不高，因为一般芯片内置的 MAC 会有网络加速引擎，比如网络专用 DMA，网络处理效率会很高。而且此类芯片网速都不快，基本就是 10/100M
### (SOC+MAC) + PHY
即`SOC`集成了网络`MAC`外设

内部集成网络 MAC 的优点如下：
- 内部 MAC 外设会有专用的加速模块，比如专用的 DMA，加速网速数据的处理。
- 网速快，可以支持 10/100/1000M 网速。
- 外接 PHY 可选择性多，成本低。

内部的 MAC 外设会通过相应的接口来连接外部 PHY 芯片，根据数据传输模式不同，大致可以分为以下两类：
- MII/RMII 接口(Media Independent Interface)：支持 10Mbit/s 和 100Mbit/s 数据传输模式；
- GMII/RGMII 接口(Gigabit Media Independent Interface)：支持 10Mbit/s、100Mbit/s 以及 1000Mbit/s 数据传输模式。
从这里可以知道，MII/RMII 接口最大传输速率为 100Mbit/s，而 GMII/RGMII 接口最大传输速率可达 1000Mbit/s

主控 SoC 需要配置或读取 PHY 芯片，也就是读写 PHY 的内部寄
存器，所以还需要一个控制接口，叫做 MIDO，MDIO 很类似 IIC，也是两根线，一根数据线叫做 MDIO，一根时钟线叫做 MDC

## PHY芯片详解
`PHY`芯片是`IEEE802.3`规定的一个标准模块
