# 依赖
## 头文件
- `include <linux/init.h>`
- `include <linux/module.h>`
# 功能
## 驱动注册
- `static int __init *_init(void);`
- `module_init(*_init)`
## 驱动注销
- `static void __exit *_exit(void);`
- `module_exit(*_exit)`
## 指定许可证
- `MODULE_LICENSE("...")`
	- `GPL`
