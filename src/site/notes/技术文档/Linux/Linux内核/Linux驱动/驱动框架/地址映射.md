# 物理地址映射为虚拟地址
## 步骤
- 定义虚拟地址指针, `eg. static void __iomem *ptr`
- 通过函数`ioremap` 映射, 参考[[技术文档/Linux/Linux内核/源码结构/arch/arm/include/asm/io.h#ioremap\|io.h#ioremap]]
- 使用完之后, 通过函数`iounmap`解映射, 参考[[技术文档/Linux/Linux内核/源码结构/arch/arm/include/asm/io.h#iounmap\|io.h#iounmap]]
