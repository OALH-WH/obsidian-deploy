## 匹配设备成功后设备初始化
**`probe`**
- 这个回调函数接受一个平台设备结构体的指针, 填充这个结构体由平台总线框架实现
	- 包括这个设备匹配到的驱动, 为`->dev->driver`
- 通过解析这个平台设备结构体的内容可以获取设备树资源, 框架提供了一些解析函数, 参考[[技术文档/Linux/Linux内核/源码结构/include/linux/platform_device.h#解析平台设备结构体\|platform_device.h#解析平台设备结构体]]
- 根据获取到的资源初始化设备
