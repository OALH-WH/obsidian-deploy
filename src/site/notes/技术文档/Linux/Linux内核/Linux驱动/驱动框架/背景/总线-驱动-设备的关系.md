# 驱动
## 传统设备驱动
主机驱动依赖不同平台上的不同资源, 因此每个平台一份驱动

但设备驱动也需要针对不同平台编写多份, 就显得有些臃肿

# 总线
管理设备与驱动的匹配，定义通用的总线级操作（如枚举、电源管理），维护该总线下的设备链表和驱动链表

# 设备
描述一个具体的硬件设备，包含物理位置、资源（寄存器地址、中断号）、设备名等信息

# 总线, 驱动和设备模型
当向系统中注册一个驱动时, 系统就会检查所有的设备是否存在与之匹配的设备

当向系统中注册一个设备时, 系统就会检查所有驱动是否存在与之匹配的驱动
