1. source insight

1. vscode

1. 配置头文件搜索路径

1. 项目目录下创建.vscode文件夹

1. 编辑c_cpp_properties.json文件

1. 配置includePath和defines

1. includePath：来指定头文件的搜索路径

1. defines：来定义宏

1. 示例：

```python
{
    "configurations": [
        {
            "name": "Win32",
            "includePath": [
                "${workspaceFolder}/**", // 包含工作区的所有文件
                "C:/Program Files (x86)/Microsoft Visual Studio/2019/Community/VC/Tools/MSVC/14.28.29333/include/**", // Visual Studio 的头文件路径
                "C:/Path/To/Your/Project/Libraries/**" // 项目中 libraries 目录的路径
            ],
            "defines": [
                "_DEBUG",
                "UNICODE",
                "_UNICODE"
            ],
            "cStandard": "c11",
            "cppStandard": "c++17",
            "intelliSenseMode": "windows-msvc-x64"
        }
    ],
    "version": 4
}
```
