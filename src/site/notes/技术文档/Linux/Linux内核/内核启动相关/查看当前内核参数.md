# 方式
## 通过sysctl命令查看
- `sysctl -a`
### 注意
- 命令不一定有
## 通过dmesg查看
- `dmesg | grep "Command line"`
### 注意
- 不一定能查到
## 通过/proc/cmdline查看
- `cat /proc/cmdline`
### 注意
- 只能查看启动参数
