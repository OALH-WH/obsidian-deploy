# 背景
- 解决内核驱动模块加载时被识别为`tainted`
# sign-file
- 必须在源码根目录下执行
## 基本格式
```shell
Usage: scripts/sign-file [-dp] <hash algo> <key> <x509> <module> [<dest>]
       scripts/sign-file -s <raw sig> <hash algo> <x509> <module> [<dest>]
```
## 参数
- `hash algo`: 哈希算法, 常用的有
	- `sha256`
	- `...`
- `key`: 私钥文件(.pem)
	- 内核源码的 `./certs/signing_key.pem`
- `x509`: x509证书文件(.x509)
	- 内核源码的 `./certs/signing_key.x509`
- `mokdulke`: 要签名的内核模块
## 实例
- `./scripts/sign-file sha256`
