# 构建
根目录执行`make dtbs`
根目录执行`make *.dts`, 指定编译某个设备树文件
## 依赖
- 需要保证自己的配置(`.config` 或者 默认配置)中存在设备树选项, 参考[[技术文档/Linux/Linux内核/源码结构/arch/arm/boot/dts/Makefile\|技术文档/Linux/Linux内核/源码结构/arch/arm/boot/dts/Makefile]]
### DTC, 设备树编译器
路径为`scripts/dtc/dtc`

| 参数       | 全称                | 含义                                          |
| -------- | ----------------- | ------------------------------------------- |
| **`-I`** | **Input format**  | 输入文件的**格式**：`dtb`（二进制）、`dts`（文本）、`fs`（文件系统） |
| **`-O`** | **Output format** | 输出文件的**格式**：`dtb`（二进制）、`dts`（文本）、`asm`（汇编）  |
| **`-o`** | **output file**   | 输出文件的**路径**                                 |
#### **实例**
**反编译**
```bash
dtc -I dtb -O dts -o atk-final.dts atk.dtb
```
