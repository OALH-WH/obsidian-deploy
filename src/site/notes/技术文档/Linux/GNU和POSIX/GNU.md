1. 目标

1. 构建一个完全自由，类似Unix的系统

1. 实现

1. glibc // GNU C Library， 实现了C/POSIX API

1. bash // GNU shell， 比POSIX sh功能强很多

1. 基本命令 // 例如ls，cp，echo等

1. gcc // GNU编译器工具链

1. 注意

1. GNU=一套实现，包括兼容POSIX标准

1. GNU是一个基于linux内核的项目, 包含LInux核心工具和组件

1. 而debian和Red Hat都是基于GNU/Linux的操作系统发行版, 相当于在GNU的基础上又增加了许多功能和软件的可用的系统
