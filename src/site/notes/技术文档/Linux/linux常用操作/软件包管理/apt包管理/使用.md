# 常见命令
## `apt`命令
---
## `apt-file`命令
### 依赖
- `apt install apt-file`: 默认不安装此命令
### 常见参数
- `search <CMD PATH>`:根据命令可以反推出包名, 推荐使用 `bin/<CMD NAME>`匹配
	- `eg. apt-file search /bin/gcc -> make: /usr/bin/make`
	- `eg. apt-file search /sbin/showmount -> nfs-common: /sbin/showmount`
---
## `apt-cache`命令
### 常见参数
- `search <package name key>`: 通过包名关键词搜索包 
---
## `apt-get`
### 常见参数
- `--allow-unauthenticated`: 跳过对软件包GPG签名的验证
	- 当你从某个源下载的软件包缺少有效签名时, 它可以让你继续安装
-  `install`: 安装软件包
	- `-y`: 安装软件包时自动确认
	- `--no-install-recommends`: 不安装推荐的软件包
