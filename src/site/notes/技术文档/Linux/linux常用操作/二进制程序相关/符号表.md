# 最关键的符号类型说明:

| 类型          | 含义                                                        | 常见于                       |
| ----------- | --------------------------------------------------------- | ------------------------- |
| **T**、**t** | 代码段(`.text`)中的**函数**。大写`T`表示全局（可被其他文件链接），小写`t`表示局部（静态函数）。 | 你编写的函数。                   |
| **D**、**d** | 已初始化的数据段(`.data`)中的**全局/静态变量**。大写`D`表示全局。                 | `int global_var = 10;`    |
| **B**、**b** | 未初始化的数据段(`.bss`)中的**全局/静态变量**。                            | `static int buffer[100];` |
| **U**       | **未定义**的符号。表示这个文件引用了该符号，但**定义在别处**（例如其他目标文件或库中）。          | `printf`（来自C标准库）。         |
| **A**       | 绝对地址符号，值在链接时不会改变。                                         |                           |
| **R**、**r** | 只读数据段(`.rodata`)中的符号。                                     | 字符串常量。                    |
| **C**       | 公共符号。一种特殊的未初始化全局数据。                                       | 某些编译器生成的旧式符号。             |
| **W**、**w** | 弱符号。即使未定义，链接也不会报错，会被默认值替代。                                | 可被覆盖的库函数。                 |
| **I**       | 间接引用（通常是动态链接的符号）。                                         | 动态库中的函数。                  |
# 示例1
```shell
0000000000401126 T main
                 U printf
                 D global_var
```
它主要由三部分（或四部分）组成：
1. **地址/值**：`0000000000401126`（符号在内存或段内的偏移地址）。如果符号是“未定义”的，则显示为空格。
2. **符号类型**：一个**关键字母**，定义了符号的属性和所在段。
3. **符号名称**：`main`、`printf`、`global_var`
