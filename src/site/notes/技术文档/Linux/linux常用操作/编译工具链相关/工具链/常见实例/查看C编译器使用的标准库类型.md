# 方式1: 通过查看编译器内部配置查看C库完整路径
## 步骤
- 从编译器内部配置中检索 `libc.so`, 参考[[技术文档/Linux/linux常用操作/编译工具链相关/工具链/编译器的使用/GCC_G++#查看C使用的标准库信息\|GCC_G++#查看C使用的标准库信息]]
- 根据返回的完整路径判断
	- 如果路径包括 `libc`目录, 就是用的 `glibc`库
