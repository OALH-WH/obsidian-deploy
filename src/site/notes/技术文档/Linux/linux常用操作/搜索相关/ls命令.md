# 背景
## 权限的作用
### 对普通文件的含义
- **r**：可以读取文件内容
- **w**：可以修改文件内容
- **x**：可以作为可执行文件运行
### 对目录的含义：
- **r**：可以列出目录内容（ls）
- **w**：可以在目录中创建、删除、重命名文件
- **x**：可以进入目录（cd）或访问目录中的文件
# 参数
- `-l`: 以列表的形式显示文件和它的一些信息, 包括权限, 文件所有者, 大小, 创建时间等信息
- `-a`: 显示所有文件, 包括`.`开头的隐藏文件
# 内容解析
## 列
- `column1`: 文件类型和权限
	- 格式: <文件类型: char1><文件所有者权限: char2-4><用户组权限: char5-7><其他用户权限: char8-10>
		- 文件类型
			- `d`: 目录
			- `l`: 符号连接
			- `b`: 块设备文件
			- `c`: 字符设备文件
			- `p`: 管道(pipe)
			- `s`: 套接字(socket)
		- 权限
			- `r`: 读权限
			- `w`: 写权限
			- `x`: 执行权限
			- `-`: 无权限
- `column2`: 文件所有者
- `column3`: 文件所在组
- `column4`: 文件大小
- `...`
- `end column`: 文件名
