# 背景
- 传输性能的最核心指标之一, 直接决定系统的误码率
- 信号再传输中会衰减, 必须用光纤放大器来放大信号
- 放大器会引入噪声, 噪声会随着信号一起被后续的放大器不断放大和累计
- 噪声会淹没信号, 随着传输距离增加, ASE(?)噪声不断累计, 而信号功率受非线性效应等限制不能无限提高。当噪声功率大到一定程度时，接收端就无法从“信号+噪声”的混合光中准确识别出原始信号，导致误码
	- **ASE噪声**: 一种因物理原因必然产生的一种宽谱噪声
		- **不是**来自外部干扰，而是**放大器固有的**。
		- 会随着信号一起在链路中**逐级放大和累积**。
		- 是导致 **OSNR劣化**，从而最终限制光通信系统**传输距离与容量**的**最核心物理机制**
- OSNR越低(噪声相对信号越强), 误码率就越高, 系统性能就越差
