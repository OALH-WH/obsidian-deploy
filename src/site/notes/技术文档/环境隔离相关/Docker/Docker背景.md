# 对比

|特性|**Docker 容器**|**传统虚拟机**|
|---|---|---|
|**操作系统内核**|**共享**宿主机内核|**独立**的完整内核（Guest OS）|
|**启动时间**|毫秒级（瞬间）|分钟级（启动内核+系统服务）|
|**资源占用**|极低（只有进程开销）|很高（占用内存、磁盘存储完整系统）|
|**兼容性**|必须与宿主机**内核版本兼容**|几乎**任何**操作系统（Windows/Linux任意发行版）|
|**隔离级别**|进程级隔离（弱隔离，共享内核有风险）|硬件级隔离（强隔离，更安全）|
