# `la_group.cpp:la_group::process()`
## 默认参数
- `suppress_src = 0`
## 函数体
- `suppress_src > 1`, 需要抑制
- 遍历组内所有告警(不包括子组告警)
	- 检查告警为 `LA_FLAG_ACTIVE`
		- `active_count++`: 统计告警应该上报的数量
		-  `active_count == 1`时获取组内第一个应该上报的告警 `index`
- 关系为 `OR`
	- 应该上报的告警数量 `> 1`: 有任何一个告警需要上报
		- `should_suppress = 1`: 应该抑制
- 关系为 `AND`
	- 应该上报的告警数量 `=`组内告警总数: 组内全部告警都要上报
		- `should_suppress = 1`: 应该抑制
- `suppress_src == 0`, 抑制的告警源为组内第一个应该上报的告警
- 抑制
	- 执行抑制控制 `la_group.cpp:la_group::suppress_ctrl(1, 0, suppress_src)`, 参考[[#`la_group.cpp la_group suppress_ctrl()`]]
- 不抑制
	- ...
# `la_group.cpp:la_group::suppress_ctrl()`
- 遍历所有子组, 并执行各自组的 `suppress_ctrl(suppress, 1, suppress_src)`:  `suppress`和 `suppress_src`都是当前函数的参数
-  `if suppress_self`, 把组内所有告警标志增加 `LA_FLAG_SUPPRESSING`, 并设置 抑制源 `suppress_src`
