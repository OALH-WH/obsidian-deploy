# 背景
- OA卡最基本的作用，就是在不经过光电转换的情况下，**直接对传输链路中的光信号进行放大**，以补偿光缆、器件等带来的功率损失，从而延长无电中继的传输距离

# 类型
根据它在网络中的位置，主要扮演三种角色：
- **功率放大器（BA，Booster Amplifier）**：安装在发射端，负责将合波后的信号增强到足够高的功率，再送入光纤线路进行长距离传输。
    
- **线路放大器（LA，Line Amplifier）**：安装在长途传输的中间节点（中继站），负责补偿光信号在光纤中传输累积的衰减。
    
- **前置放大器（PA，Preamplifier）**：安装在接收端，在信号被分波或检测之前，对已经非常微弱的信号进行预放大，以提高接收灵敏度。
