# Web3
## 背景
- `Web3.0`采用`Web2.0`的交互方式, 但更改了底层结构, 使其更加公平, 公开, 和去中心化 
### Web1.0
- 万维网
- 一个网站里没有太多交互性, 数据和内容都来自静态文件(.html)
- 网站都能向想要了解它的用户提供信息-仅此而已
### Web2.0
- 互联网
- 动态`HTML`, 可以和其他用户交互
- 缺点
	- 数据完全保存在具有盈利效应公司手上的服务器上, 缺乏数据隐私
	- 数据过于集中, 大部分数据都保存在服务器上
