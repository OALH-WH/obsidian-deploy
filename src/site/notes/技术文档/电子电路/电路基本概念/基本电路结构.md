# 背景
- 电压: 电路中两端的电势差, 就叫电压
- 电流: 电路中的一个横截面, 电子流过的速度
- 欧姆定律
- 电势越低，负电荷越多（电子越多）。
- 电势越高，正电荷越多（电子越少）。
- 电子带负电，总是被吸引到正极。
# 短路
- 电流几乎全部选择阻值更小的路径流过, 其他设备不工作 
# 断路
- 电路中没有形成回路
# 串联
- 如果拿掉串连的设备, 则无法形成通路
- 串联分压
# 并联
- 拿掉并联的设备之后, 电路仍旧是一个通路
- 并联分流, 并联设备的总阻值变小
# 实例
## 跳闸, 保险丝熔断
- 一般是并联电路导致的, 并联设备变多, 电阻变小, 电压不变, 把并联设备当作一个整体, 流过它们的电流会变大, 超过导线能承载的最大电流, 导致保险丝熔断
