# 命令面板
## 清除终端缓存
- `terminal clear`: 键入关键字
# 配置
## settings
- `terminal.integrated.scrollback`: 终端缓存行最大长度
### 实例
```json
"terminal.integrated.profiles.windows": {
    "Git Bash": {
        "path": "C:\\Program Files\\Git\\bin\\bash.exe",
        "icon": "terminal-bash"
    }
},
"terminal.integrated.defaultProfile.windows": "Git Bash"
```
