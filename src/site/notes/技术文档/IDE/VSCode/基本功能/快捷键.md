# 配置
## 进入方式
### Shift+Ctrl+p: 键入keyboard shortcuts(JSON)

## 配置实例
### Json方式内容解析
- `"key": "SHORTCUT"`: 表示快捷键
- `"command"`: 表示行为
	- `"-<ACTION>"`: 表示删除某个快捷键
- `when`: 表示触发条件
### 关闭Esc默认退出Vscode面板
#### 解决示例
- 比如关闭默认行为: 键入Esc退出终端面板
#### 配置方法
#### JSON方式
```json
# 添加如下内容
{
  "key": "escape",
  "command": "-workbench.action.closePanel",
  "when": "panelFocus"
}
```
