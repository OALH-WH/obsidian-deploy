#  Cortex-Debug插件
- 单片机调试插件
## 配置
### launch.json
#### 属性字段
参考<https://gitcode.com/gh_mirrors/co/cortex-debug/blob/07b30999fb55601778fc1de47a10460f1be39221/debug_attributes.md?utm_source=gitcode_repo_files&isLogin=1&from_link=e8c4685a6bdfa23f967860116f6ca819>
- `servertype`: `gdb-server`的类型
	- `external`: 使用外部`gdb-server`, 即假设由用户自定义`gdb-server`的启动, 但需要和属性`gdbTarget`配合使用
- `gdbTarget`: 当`servertype`为`external`时, 指定`gdb`要连接的`IP:PORT`
	- `eg. localhost:3333`
