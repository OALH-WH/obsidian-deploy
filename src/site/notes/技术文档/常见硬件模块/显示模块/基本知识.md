### 接口类型
**RGB屏/DPI接口屏**
一种不带显示控制器的`LCD`面板, 在这种模式中，`LTDC`通过`MCU`引脚直接与`RGB`屏连接，将并行的`RGB`信号直接传给屏幕显示。

**DSI接口屏**
MIPI-DSI并非一个独立的外设，而是在LTDC外部集成了一个专用的**DSI主机（DSI Host）**[](https://www.alldatasheet.jp/html-pdf/933979/STMICROELECTRONICS/STM32F779IGT6/101706/52/STM32F779IGT6.html)。两者关系非常紧密：
- **数据流向**：LTDC生成并行RGB视频信号，然后内部传给DSI主机。
- **核心功能**：**DSI主机的作用是一个“协议转换器”，它将LTDC输出的并行RGB信号，编码为高速的、串行的MIPI DSI差分信号，再通过少量引脚输出**[](https://www.alldatasheet.jp/html-pdf/933979/STMICROELECTRONICS/STM32F779IGT6/101706/52/STM32F779IGT6.html)。
- **关键约束**：它们之间是“紧耦合”关系，**不能绕过LTDC直接使用DSI接口**。配置时，**必须确保LTDC的输出像素率，不能超过DSI接口的传输带宽上限**。
这意味着，DSI接口是依赖LTDC才能工作的，它更像一个后端的“编码发送器”。

**HDMI接口屏**
HDMI接口又是另一回事了。绝大多数MCU内部都**没有集成**HDMI控制器。因此，要实现HDMI输出，需要以下两种方法：

- **方法一：外部桥接芯片（最常见）**：在PCB上增加一颗独立的桥接芯片（如SII9022[](https://wiki.i2som.com/docs/panguboarddoccn/panguboarddoccn-1fthpjat18p3r)）。此时，LTDC将RGB信号输出给这颗芯片，由它将RGB信号“翻译”成HDMI信号输出。**如果LTDC不支持某分辨率，HDMI也无法支持**[](https://wiki.i2som.com/docs/panguboarddoccn/panguboarddoccn-1fthpjat18p3r)。
    
- **方法二：经过DSI中转**：流程变为 `LTDC -> DSI Host -> MIPI-DSI信号 -> 外部MIPI-DSI转HDMI芯片 -> HDMI信号`，链路更长。



# 色彩深度
**色深**，也被称为**位深**，指的是图像或显示设备中，用于表示**每个像素**颜色或亮度信息的**二进制数据位数**。色深直接决定了可显示颜色的总数量，而颜色的总数量又决定了颜色之间过渡的级别数量
- **单位：** 位/像素。
- **本质：** 它决定了每个像素能呈现多少种不同的颜色或灰度级别。
- **计算公式：** 可表现的色彩/灰阶数量 = 2 的“色深”次方。
**举个例子：**
- **1位色深：** 2¹ = 2种颜色（通常是纯黑和纯白）。
- **8位色深：** 2⁸ = 256种颜色或灰度级别。

**单色**, 单色并不等于黑白，而是指在一种基色下，拥有从暗到亮的无数个层次。
**灰度**, 单色模式的一个标准特例，特指没有彩色信息，纯用不同亮度的灰色（从黑到白）来构成的图像。
# 刷新率
主要取决于屏幕的渲染周期, 比如`60fps`表示`1s`内刷新`60`次
# 分辨率
单位面积内, 能容纳的像素点数量, 一般用显示器总像素数量描述, 比如`1092*960`
# 对比度
画面中明暗区域最亮的白和最暗的黑之间不同亮度层级的差异
## 视觉上：清晰与模糊的差别
- **高对比度**：
    - **感觉**：画面鲜亮、锐利、轮廓分明、有冲击力。
    - **特点**：亮部很亮，暗部很暗，黑白分明，中间过渡的灰色调较少。
    - **例子**：阳光下物体的清晰影子；一张文字洁白、背景纯黑的文档。
- **低对比度**：
    - **感觉**：画面柔和、朦胧、灰蒙蒙、平淡，甚至有点“脏”。
    - **特点**：最亮和最暗的区域差距小，很多灰色调，没有纯黑或纯白。
    - **例子**：大雾天的风景，所有东西都若隐若现；一张浅灰文字配深灰背景的图片，看起来费劲。

# 像素格式
## RGB888
一般一个像素点的 `R、G、B` 这三部分分别使用 `8bit` 的数据，那么一个像素点就是` 8bit*3=24bit`，也就是说一个像素点 `3`个字节，这种像素格式称为 `RGB888`

## RGB8888
再加入 8bit 的 Alpha(透明)通道的话一个像素点就是 32bit，也就是 4 个字节，这种像素格式称为 ARGB8888
