# 其他
- `npx clawhub@latest install <技能名称>`: 从clawhub上下载skill
	- 相当于使用`npx`运行clawhub
# npx子命令

## skills
- `find [query]`: 关键字搜索技能
- `add <package>`: 从GitHub或者其他来源安装技能
- `check`: 检查技能更新
- `update`: 更新所有安装的技能
