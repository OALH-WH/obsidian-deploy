# C/C++
- 科学计数法
	- 常见格式
		- `%[.precision]e`: 指数用小写`e`
		- `%[.precision]E`: 指数用大写`E`
	- 说明
		- `.precision`: 精度
- 浮点数控制
	- 格式
		- `%[flags][width][.precision]specifier`
	- 说明
		- `flags`: 左对齐, 显示正负号等
		- `width`: 表示总宽度, `eg. %11.5f`表示总宽度11个字符
		- `.precision`: 精度, 表示精确到小数点后几位, `eg. %11.5f`表示精确到小数点后5位
		- `specifier`: 说明符, `f`表示浮点
