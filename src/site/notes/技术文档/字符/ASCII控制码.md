# 背景
## C 的 stdout 有三种缓冲模式

| 模式                  | 什么时候发生               | 模式行为                                        |
| ------------------- | -------------------- | ------------------------------------------- |
| 全缓冲（block buffered） | 默认（`stdout`不是 `TTY`） | 程序结束, 或者手动执行`fflush()`, 才会一次性输出到`stdout`    |
| 行缓冲（line buffered）  | `stdout` 是 `TTY`     | 检测到`\n`, 即换行, 才会刷新内容到`stdout`               |
| 无缓冲                 | stderr / 手动设置        | 设置示例`eg. setvbuf(stdout, NULL, _IONBF, 0);` |
|                     |                      |                                             |
# 其他转义字符
1. \t：水平制表符，即tab按键
2. \n: 换行符
- `\r`: 回车符, 即把光标移动到行首
	- 生效条件
		-  **字符真的被写到了终端（TTY）**
		- **写入是“实时”的（没有被 stdio 缓冲吞住）**
		- **终端模拟器按控制字符解释它**
1. \b: 退格符，光标回退到上一个字符的位置
# 颜色转义字符
- 颜色顺序: 黑、红、绿、黄、蓝、紫、深绿、白
- `\e == \033`
- `\e[0m`: 清除颜色代码
- `\e[30m`: 黑色前景
- `\e[40m`: 黑色背景
# 控制转义
- 上移n行：`\e[nA`
- 下移n行：`\e[nB`
- 右移n列：`\e[nC`
- 左移n列：`\e[nD`
## 清除当前行的内容: `\e[nK`
### n 的取值与含义（非常重要）

|n|含义|
|---|---|
|`0`|从**光标位置**清到**行尾**（默认）|
|`1`|从**行首**清到**光标位置**|
|`2`|**清空整行**|
