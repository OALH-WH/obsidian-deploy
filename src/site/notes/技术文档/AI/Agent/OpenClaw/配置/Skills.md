## npx skills包管理器

## `npx clawhub@latest` 命令
- 参考[[技术文档/node.js/包管理器/npx命令\|npx命令]]
- `eg. npx clawhub@latest search <key word>`: 在`clawhub`上关键词搜索`skill`
- `eg. npx clawhub@latest install <skill name>`:  安装`skill`
- `eg. npx clawhub@latest list`: 查看所有已经成功安装的`skill`
