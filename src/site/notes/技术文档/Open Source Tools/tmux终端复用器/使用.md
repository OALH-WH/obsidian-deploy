## tmux空间相关
### 彻底退出 tmux 服务器（结束所有会话）

如果你想一次性结束所有 tmux 会话并退出 tmux 服务：

- 在终端中输入：`tmux kill-server`  
    （无需在 tmux 内部执行，直接在普通终端执行即可）

## 会话相关
### 重命名会话
- 快捷键
	- `$`
- 命令
	- `tmux rename-session <new-name>`: `tmux`内执行
	- `tmux renmae-session -t <old-name> <new-name>`: `tmux`外执行
### 分离会话
- 快捷键
	- `d`
- 命令
	- `tmux detach`
### 杀掉一个会话
- 命令
	- `tmux kill-session -s <会话名称>`: 删除关闭一个会话
### 创建一个会话
- 命令
	- `tmux new -s <会话名称> [执行的命令]`: 开启一个会话\[执行某些命令后直接结束会话]
	- `tmux new-session  -s <会话名称> [执行的命令]`: 上同
### 检查会话是否存在
- 命令
	- `tmux has-session -t <会话名称>`: 可通过$?是否为0判断会话是否存在, 会话名称参考 `tmux ls`

## 窗格相关
### 划分窗格
- 命令
	- `tmux split-window`: 上下划分
	- `tmux split-window -h`: 左右划分
- 快捷键
	- `"`: 上下划分
	- `%`: 左右划分
### 其他
- `tmux list-commands`: 查看所有支持的命令

# 注意
- tmux的会话只要有一个没关闭, 就会沿用它的shell环境, 因此对.bashrc这种有操作, 要关闭所有会话才能默认使用修改后的shell环境
