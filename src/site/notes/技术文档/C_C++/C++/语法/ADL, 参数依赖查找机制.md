# 背景
- ADL机制，Argument-Dependent Lookup, 参数依赖查找
# 机制逻辑
## 常规查找
- 在函数调用点所在的命名空间查找
	- 示例
		- 使用`std::to_string()`, 所以它首先会在std命名空间中找到所有名为`to_string`的函数
## 参数依赖查找
- 对于函数参数为自定义类型, 比如**自定义结构体/类**, 编译器在完成常规查找后, 会自动的, 隐式的将**自定义结构体/类**所在命名空间加到查找范围里
