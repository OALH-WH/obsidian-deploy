# 类
## 构造函数的初始化列表
### 初始化列表基本格式
```cpp
 Derived(int id, const std::string& n, double v)
        : Base(id),      // 1. 先初始化基类
          name(n),       // 2. 再初始化成员变量（按声明顺序）
          value(v)       // 3. 继续初始化其他成员
{
	// 构造函数体
	std::cout << "Derived 构造函数, name=" << name 
			  << ", value=" << value << std::endl;
}
```
### 初始化列表写法总结表

| 初始化项         | 写法示例                  | 是否可选                | 注意事项          |
| ------------ | --------------------- | ------------------- | ------------- |
| **直接基类**     | `: Base(参数)`          | **必须**（除非基类有默认构造函数） | 按继承顺序初始化      |
| **成员变量**     | `: 成员名(值)`            | 可选                  | 按类中声明顺序初始化    |
| **const 成员** | `: const_member(值)`   | **必须**              | 必须在初始化列表中初始化  |
| **引用成员**     | `: ref_member(变量)`    | **必须**              | 必须在初始化列表中初始化  |
| **类类型成员**    | `: object_member(参数)` | 可选（会调用默认构造函数）       | 如果没有默认构造函数则必须 |
