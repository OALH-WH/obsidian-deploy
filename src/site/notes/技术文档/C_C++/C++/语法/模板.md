# 函数模板
## 模板参数
### 模板参数推导
**示例**
```cpp
template<typename T>
func(T&& a);

int main(){
	int a;
/*
T被推导为int& -> T&& a=int&&& a
经过引用折叠得到int& a	
*/
	func(a); 
	
/*
T被推导为int&& -> T&& a=int&&&& a
经过引用折叠得到int&& a	
*/
	func(12);
}
```

## 可变模板参数
### 核心概念
- ‌**参数包**‌（Parameter Pack）：用 `...` 声明，表示0到多个模板参数或函数参数。
- ‌**包展开**‌（Pack Expansion）：通过 `...` 在右侧将参数包拆解为独立参数。
- ‌**`sizeof...`**：编译期运算符，用于获取参数包中参数个数, `eg. sizeof...()`
- 
### 展开方式
#### 递归展开
- 两种递归终止函数: 一种带参, 一种不带参
```cpp
// 终止函数
void print_all() {
    std::cout << std::endl;
}

// 递归展开
template<typename T, typename... Args>
void print_all(T first, Args... rest) {
    std::cout << first << " ";
    print_all(rest...);  // 展开剩余参数
}

// 外部接口
template<typename... Args>
void Print(Args... args) {
    print_all(args...);
}

```
#### 逗号表达式+初始化列表
```cpp
template<typename T>
int PrintArg(T val) {
    std::cout << val << " ";
    return 0;
}

template<typename... Args>
void Print(Args... args) {
    int arr[] = { PrintArg(args)... };  // 展开为 { PrintArg(arg1), PrintArg(arg2), ... }
    std::cout << std::endl;
}

```
