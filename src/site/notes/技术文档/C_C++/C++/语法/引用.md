# 值的类型
## 左值
左值是表达式的一种属性，表示可以出现在赋值运算符左侧的值
## 右值
右值是指不能出现在赋值运算符左侧的表达式，通常表示临时对象或即将被销毁的对象
# 引用类型
## 左值引用
```cpp
int &a;
```
## 右值引用
```cpp
int &&b;
```
## 万能引用
```cpp
func(int &&c); //只会出现在形参定义中
```
