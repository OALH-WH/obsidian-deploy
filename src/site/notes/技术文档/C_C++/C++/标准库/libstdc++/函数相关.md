# 函数
## std::bind
- 用于对一个函数绑定参数, 生成一个新的有缺省参数的函数
### 示例
```cpp
auto add10 = std::bind(add, std::placeholders::_1, 10);

int r = add10(5);  // 结果 15

auto handle_notification_test = std::bind(&Lai::handle_notification, this, _1, _2)

handle_notification_test(.., ...) //对于实例方法, 要加this, _1, _2为占位符, 表示调用时arg1, arg2在实际调用函数时的参数位置
```
