1. 注意

1. 特化必须在隐式对应类型的实例化首次使用之前

1. 只有声明没有定义的模板特化可以像使用其他不完整类型一样使用，例如可以使用到它的指针和引用

1. 函数模板和变量模板的特化是否为inline/constexpr/constinit/consteval只与显式特化自身有关，主模板的声明是否带有对应说明符对它没有影响， 如果主模板有的，但是特化模板没有，则以特化模板为主

1. 模板全特化要在前面加template <>

1. 告诉编译器，这是一个特化版本，而不是定义一个新的模板版本

1. 偏特化不用写template <>

1. 写了的话，编译器会认为没有模板参数了，但实际上偏特化还有模板参数

1. 函数模板

1. 全特化

1. template <> func_name<actual_type1, actual_type2...>(actual_type1 arg1, actual_type2 arg2...)

1. 类模板

1.  全特化

1. template <> class my_class<actual_type1>

1. 变量模板

1. 可以通过特化来实现给变量赋予不同的值

1. 示例

```cpp
template<typename T>
constexpr bool is_void_v = false;

template<>
constexpr bool is_void_v<void> = true;

int main(){
    std::cout << std::boolalpha << is_void_v<char> << '\n';   // false
    std::cout << std::boolalpha << is_void_v<void> << '\n';   // true
}
```
