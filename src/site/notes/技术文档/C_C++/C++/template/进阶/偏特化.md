1. 概念

1. 让模板实参具有一些相同特征可以自定义，而不是像全特化那样必须是具体的什么类型，什么值

1. 变量模板偏特化

```cpp
template<typename T>
const char* s<T*> = "pointer";  // 偏特化，对指针这一类类型

template<typename T>
const char* s<T[]> = "array";  // 偏特化，但是只是对 T[] 这一类类型，而不是数组因为 int[] 和 int[N] 不是一个类型
```

1. 类模板偏特化

```cpp
// 简化理解（C++11标准库实现风格）
template <typename T, typename U>
struct is_same {
    static constexpr bool value = false;
};

// ✅ 特化版本：当 T 和 U 完全相同时
template <typename T>
struct is_same<T, T> {
    static constexpr bool value = true;
};

```
