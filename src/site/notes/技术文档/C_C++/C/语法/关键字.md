# goto
## `feature`
- 跳转到标号处开始执行代码
## `note`
- 只能在同一函数内跳转, 不能跨函数
- 在`C++`中, 不能跳过有初始化的变量
	- 会破坏代码结构
	- 下文可能使用被跳过的未初始化的变量
- 当跳过某个完整的作用域时, 作用域中初始化变量的行为可以被接受
	- 因为跳过作用域之后, 即使其中变量没有初始化, 跳过后的代码也无法使用未初始化的变量, 因此是安全的
# typedef
## `feature`
- 定义类型别名
## `example`
### 定义函数指针类型别名
```c
typedef int (*test_type_t)(void);
test_type_t func_ptr;
```
