1. GCC

1. 关键字：__asm__ 或__asm__ __volatile__

1. 示例

```cpp
 __asm__ (
        "addl %1, %0"  // 汇编指令
        : "=r" (result)  // 输出操作数
        : "r" (a), "r" (b)  // 输入操作数
    );
```
