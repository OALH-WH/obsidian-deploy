# 数据访问宽度控制
## 原理
- 在ARM Cortex-M等处理器中，CPU通过**内存访问指令**（如LDRB / LDRH / LDR / STRB / STRH / STR）来读写外设寄存器。C语言编译时，会根据指针所指向的类型（`uint8_t*`、`uint16_t*`、`uint32_t*`）生成对应宽度的指令。
## 读
```c
#define QSPI_DR_ADDR  0x5800400CU   // 假设QSPI_DR的地址

// 字节读：从FIFO读取1个字节
uint8_t  byte_val = *(volatile uint8_t  *)QSPI_DR_ADDR;

// 半字读：从FIFO读取2个字节（低16位有效）
uint16_t half_val = *(volatile uint16_t *)QSPI_DR_ADDR;

// 字读：从FIFO读取4个字节
uint32_t word_val = *(volatile uint32_t *)QSPI_DR_ADDR;
```
## 写
```c
*(volatile uint8_t  *)QSPI_DR_ADDR = 0xAA;    // 写入1字节
*(volatile uint16_t *)QSPI_DR_ADDR = 0xAABB;  // 写入2字节
*(volatile uint32_t *)QSPI_DR_ADDR = 0xAABBCCDD; // 写入4字节
```
