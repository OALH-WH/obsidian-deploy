1. 创建项目

1. 项目根目录下进入rt env环境

1. 执行scons --target=vsc: 创建vscode工程

1. 执行scons可以编译项目生成可执行文件

1. 把ENV工具中的ConEmu目录放到path环境变量中

1. vscode 的工作区setting.json中添加配置如下

```cpp
{
  "terminal.integrated.profiles.windows": {
    "RT-Thread Env": {
      "path": "D:\\RT-ThreadStudio\\platform\\env_released\\env\\tools\\ConEmu\\ConEmu.exe" // 替换为你的实际路径
    }
  },
  "terminal.integrated.defaultProfile.windows": "RT-Thread Env"
}

{
  "terminal.integrated.profiles.windows": {
    "RT-Thread Env": {
      "path": "C:\\Users\\OALH\\MyFiles\\IDE\\RT-ThreadStudio\\platform\\env_released\\env\\tools\\ConEmu\\ConEmu.exe" // 替换为你的实际路径
    }
  },
  "terminal.integrated.defaultProfile.windows": "PowerShell"
}
```
