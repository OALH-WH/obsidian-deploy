## 步骤
- 以下步骤以**float浮点**作为参考
- 整数转二进制表达, `eg. 13d=1101b`
- 规范化表示, `eg. 1101 = 1.101 × 2^3`
- 计算IEEE 754 各部分值
	- 符号位, `eg. 正数 -> 0`
	- 指数位, `eg. 3 + 127(8位偏移量) = 130d = 10000010b`
	- 尾数位, `eg. .101 = 10100000000000000000000b`
- 组合, `eg. 0 10000010 10100000000000000000000b`
